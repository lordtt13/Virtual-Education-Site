VERIFICATION OF KVL AND KCL

  
      

AIM:

Solve the given circuit by Mesh & Nodal analysis and verify with OrCAD PSpice software.

SOFTWARE REQUIRED:

OrCAD PSpice/MATLAB-SIMULINK

THEORY:

Circuit analysis defines as to calculate the branch current and nodal voltages of given. There are many methods for calculating these values. Using resistance reduction method with ohms law, the severity of the calculation will increase as number of loops and nodes increases in the given circuit. Mesh current and Nodal voltage analysis are two methods which uses the kirchoff's voltage and current law for calculting the branch currents and node voltages.

MESH CURRENT ANALYSIS:

Mesh current analysis is also called as Loop current analysis. This Method uses the kirchoff's voltage law to calculate the unknown currents in the branch by framing minimum loops and minimum equations.

Solution Using Matrix Method:

      

# Find out number of loops in the circuit that covers all elements in the circuit atleast once.

# Consider the clockwise flow of current direction on each loop and mark the unknown currents I1, I2, I3 and so on.

# Frame the matrix.

     [R][I] = [V]

      

# The diagonal matrices R11, R22 and R33 defines the sum of resistances in that respective loop.

# The half diagonal matrices R12, R21 and R13, R31, R23 and R32 defines the resistance appears in between the respective loops.

# The voltage matrix V1, V2, V3 defines the sum of voltages in the respective loop.(+ and - sign will come according to flow of current directions).

# The Current Matrix I1, I2, I3 defines the unknown values of loop currents.

NODAL VOLTAGE ANALYSIS:

Nodal Voltage analysis method uses the kirchoff's current law to calculate the unknown voltages at the nodes by framing minimum node equations.

Solution Using Matrix Method:

      

# Find out number of nodes in the circuit that covers all elements in the circuit atleast once.

# Set refernce node among all and that should be common to remaining other node. Mark the unknown Node Voltages V1, V2, V3 and so on.

# Frame the matrix.

     [G][V] = [I]

      

# The diagonal matrices G11, G22 and G33 defines the sum of conductances connected at that respective node(i.e., 1/R1+1/R2+1/R3+...)

# The half diagonal matrices G12, G21 and G13, G31, R23 and G32 defines the conductances connected directly between the respective nodes.

# The voltage matrix V1, V2, V3 defines the unknown node voltages in the respective node.

# The Current Matrix I1, I2, I3 defines the source currents flow towards the node or coming out from from the node.

PROCEDURE:

1. In the given circuit, Convert all the current sources into the voltage sources if possible.

2. Find out the number of closed loops.

3. Consider the clockwise current flow direction in all the loops.

4. Frame current equations on each loop using KVL and solve it.
