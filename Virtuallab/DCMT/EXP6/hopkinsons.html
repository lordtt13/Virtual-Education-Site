Thoery:

The efficiency of the dc machine can be accurately determined by the regenerative method normally known as Hopkinson’s test. This test overcomes the drawback of Swinburne’s test, which does not take any account of the stray load losses occurring in dc machines under loaded conditions. As such the efficiency, calculated by Swinburne’s test is comparatively higher than the actual one. Hopkinson’s test needs two identical dc machines coupled mechanically and connected electrically as shown in circuit diagram. One of the machines is operated as motor, driving the other machine as a generator. The output power of the generator is fed to the motor. Thus the power drawn from the supply is only to overcome the losses of both the machines. By varying the field currents of generator and motor i.e.I4 and I3, any desired load can be adjusted on these two machines. Referring the circuit diagram of figure following expressions can be established for this case.

       Let    
                         V = Supply voltage
                         I1 = Current taken from supply
                         I2 = exciting current (field current) of motor
                         I3 = Output current of generator
                         I4 = exciting current (field current) of generator
                         Ra = armature resistance of each machine
                         
            Motor input =  V ( I1 + I3 ) -----------------------------------------------------------(1)
            
            Generator output = V I3
            
            If both the machines have same efficiency η, then
            
            Output of motor     = η * Input
                                = η * V ( I1 + I3 ) = generator input
                                
            Output of generator = η * input
                                = η * η V (I1 + I3) = η2 V (I1 + I3) --------------------- (2)
                                
            Equation (1) and (2) we get
                     
                              η2 V ( I1 + I3 ) = VI3
                              η = {√ [ I3 / ( I1 + I3 ) ] }

If the efficiencies of the two machines are not equal (armature currents as well as their excitations being different) the efficiencies are found separately as follows:

            Armature copper loss in generator = ( I3 + I4 ) 2Ra
            Armature copper loss in motor =  ( I3 + I1 – I2 ) 2Ra
            Shunt copper loss in generator = VI4
            Shunt copper loss in motor = VI2
            
But total motor and generator losses are equal to the power supplied by the mains.

            Power taken from supply = VI1
            
Now, 
            total stray losses for the set = VI1 - [(I3 + I4) 2Ra + (I3 + I1 – I2) 2Ra + VI4 + VI2] = P stray
            
Assuming that stray losses are equally divided between the two machines, we have

            Stray loss per machine = P stray / 2


For generator
 

            Total losses = (I3 + I4) 2Ra + VI4 + P stray / 2
            
            Output = VI3

            Efficiency of generator ηg = Output / (Output + Losses)
                                       = VI3 / [  ( VI3 ) + ( I3 + I4 ) 2Ra +  ( VI4 ) + ( P stray / 2) ]


For motor

            Total losses = ( I3 + I1 – I2 ) 2Ra + VI2 + ( P stray / 2) 
 
                  Input  =  V ( I1 + I3 ) 

Efficiency of motor   ηm = [Input – losses] / [Input]

                         = [ V ( I1 + I3 ) ]  - [ ( I3 + I1 – I2 ) 2Ra + VI2 + ( P stray / 2) ] / [ V ( I1 + I3 ) ]

Procedure:

1. Develop a MATLAB/simulink model of given circuit diagram (Fig.1).

2. Initially motor's feild rheostat should kept at minimum and generator's feild rheostat should kept at maximum.

3. Motor has to start at low speed.

4. By adjusting the motor's feild rheostat set rated value of speed of generator.

5. By adjusting generator's feild rheostat set differential voltages of both machines are as zero. close the SPST switch.

6. Now adjust the generator field rheostat for various values of generator current I4.For various values of I4 note down the meter readings of I2, I3 and I1. For various value of I4 the rated speed should be maintained constant.

7. Repeat step 6 till the full load current of the generator.

8. Reduce the load on the generator and motor by varying their field currents. stop the simulation.
