DC SWEEP ANALYSISFOR MAXIMUM POWER TRANSFER

  
      

AIM:

Perform DC sweep analysis for identifying the value of RL to have maximum power.

SOFTWARE REQUIRED:

OrCAD-PSpice/MATLAB-SIMULINK

THEORY:

The DC Sweep analysis generates output like that of a curve tracer. This procedure allows you to simulate a circuit many times, sweeping the DC values within a predetermined range. The DC sweep analysis can be applied for many applications of electronic circuits where the electrical output parameter has to be analysed for different values of input parameter.

In this experiment, it performs a series of power measurement for different values of RL, modifying the resistance RL as said in pre-defined steps, to give a power transfer curve. From the power transfer curve, the point at which the power is maximum indicates the values of load resistance RL to have maximum power from the source connected. The value of RL obtained from the power transfer curve is also the internal resistance or thevenin's resistance of the circuit for which the load is connected.

The Maximum Power Transfer Theorem states that, " In an active network, maximum power transfer to the load resisitance when load resistance is equivalent to the internal resistance or the net resistance across the load terminals".

PROCEDURE:

1. Design the circuit using the components from the libraries of ORCAD.

2. Edit the values for each component in the circuit designed except for the load.

3. Declare a variable for the load resistance as its value in parenthesis eg: {R } where R is the variable declared for the load resistance.

4. Get into the ‘SPECIAL’ library of ORCAD and select the part ‘PARAM’ and put it near to load resistance.

5. Double click on the part ‘PARAM’ and add a new column. In the new column give the load resistance parameter name R as name and some random value as value.

6. Create the profile and in the setting the profile, select DC sweep analysis.

7. Under the options provided in the profile settings, select parametric sweep. Then, under sweep variable, select ‘Global Parameter’ and mention the parameter name same as the variable name declared for load resistance i.e. ‘R’ and give the start value and end value with the increment value for which the load resistance value has to be verified for maximum power measurement.

8. Use the wattmeter probe to measure the power across the load resistance.
