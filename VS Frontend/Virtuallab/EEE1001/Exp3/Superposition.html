AIM:

Solve the given circuit by Superposition Theorem and verify with OrCAD-PSpice software.

SOFTWARE REQUIRED:

OrCAD-PSpice/MATLAB-SIMULINK

THEORY:

If a circuit has two or more independent sources, one way to determine the value of a specific variable (voltage or current) is to use nodal or mesh analysis. Another way to determine the contribution of each independent source to the variable and then add them up. This approach is known as superposition priniciple. The superposition principle states that the voltage across (or current through) an element in a linear circuit is the algebraic sum of the voltages across (or currents through) that element due to each independent source acting alone.

To apply the superposition principle,

We consider one independent source at a time while all other independent sources are turned off. This implies that we replace every voltage source by 0 V (or a short circuit), and every current source by 0 A (or an open circuit). Dependent sources are left intact bacause they are controlled by circuit variables.

PROCEDURE:

Superposition Theorem:

1. Calculate the current flowing through each branch of given circuit by keeping only one independent source at a time.

2. Repeat the above step for remaining all other sources and calculate the current flowing in each branch because of respetive sources.

3. Adding all calculated currents of respective branches give the actual current flow of each branchs.
