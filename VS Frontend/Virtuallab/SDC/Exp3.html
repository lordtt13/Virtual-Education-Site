Perform the Characteristics of BJT

  
      

AIM:

To obtain the input and output characteristics of a bipolar junction transistor and find the h parameters for the same.

SOFTWARE REQUIRED:

OrCAD PSpice/MATLAB-SIMULINK

THEORY:

Transistor is a three layer, three terminal current operated device where the current flowing through two terminals can be controlled by the signal on the third terminal. For the proper working of the transistor the input junction should be forward biased and the output junction should be reverse biased.

The three regions are emitter, base and collector. The three regions does not have the same thickness, and are not doped to the same level. This is necessary for the transistor to be used as an active device. The emitter is heavily doped and it has a considerable thickness. The base is lightly doped and it is very thin. The collector is moderately doped but it is thicker than emitter. In NPN transistor, when the emitter-base junction is forward biased the electrons diffuse into the base region. Simultaneously a small number of holes from the base diffuse into the emitter region. Some of the electrons recombine with this hole in the base region producing a base current (IB). As the collector base junction is reverse biased, minority carriers namely electrons in the base region and holes in the collector region can drift across the reverse biased collector-base junction, thus constituting the collector current (Ic).

INPUT CHARACTERISTICS

The collector-base voltage is kept constant and the emitter-base voltage is varied. The values of the emitter-base voltage and the corresponding emitter current are tabulated for a constant collector-base voltage and the graph is plotted. From the graph find the h-parameters hfb and hib.

The input characteristic is nothing but emitter diode junction characteristics. The transistor input characteristic is essentially that of a forward biased base-emitter junction, with some dependence upon the junction CE. With VCE is set to some volts, the effective width of the base will decrease. This effect causes a decrease in the base current IB. Hence to get the same value of IB as before, VBE should be increased. Therefore the curve shifts to right.
OUTPUT CHARACTERISTICS

To determine the output characteristics, the emitter current is kept constant and the collector base voltage is varied. The values of collector-base voltage and the corresponding collector current are tabulated for a constant emitter current and the graph is plotted. From the graph find the h parameters hob and hrb.

To determine the output characteristics, the base current IB is kept constant. The output characteristics have three regions namely, saturation region, cut-off region, and active region. In saturation region both the junctions are forward biased and a increase in the base current IB does not cause a corresponding large change in Ic. The Cut-off region both junctions are reverse bias. In the active region, emitter-base junction is forward biased and the collector-base junction is reverse biased. For the transistor to be used as linear amplifier, it should be operated in the active region.
