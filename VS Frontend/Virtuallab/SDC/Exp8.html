Out Put Reading : Probe Cursor

THEORY:

V ripple is the ripple voltage obtained due to the difference in V output max V output min Across the capacitor these voltages are obtained while charging & discharging of a capacitor which is a energy storing element average of the minimum &maximum voltages obtained across the capacitor gives us the V output.

PROCEDURE:

1.We will open Orcad PSpice

2. Then we will select place ,the part then add library select all ctrl +A

3. Then we will select a source place it in the window place

4. Then we will follow step2 and select diodes of 1N4007 type place this in window space .

5. Then we will select part, then add library then we will type analog .

6.Then we will select C i.e. capacitor and place it in the window space .

7.Then we will follow step5 and then we will select a resistor and will place it in the window space.

8.Now we will select wire and will connect the circuit as shown in the circuit diagram then we will select ground .
