THEORY:

Logic gate is a circuit that can decide to say YES or NO at the output, based upon the inputs. Logic gates can be constructed by using simple switchess, Relays, Transistors and diodes or integrated circuits.

NAND gate: NAND gate is a NOT AND or an inverted AND function. This gate is a universal Gate, which is used to create any of the other basic logic functions. The truth table is shown in the diagram. Where A,B are the inputs, Y is the output.

NOR gate: The NOR gate is actually a NOT OR gate. The truth table is shown in the diagram. Where A,B are the inputs, Y is the output.

NOT gate: NOT gateis often called an inverter. This has only one input and one output. T he job of this gate is to give an output that is not the same (opposite) as the input. The truth tableis shown in the diagram. Where A,B are the inputs, Y is the output.

AND gate: AND gate is sometimes called the all or nothing gate. So the AND gate outputs "HIGH"(binary 1) at the output only if the HIGH present at he both the inputs. The Truth table is shown in the diagram the Boolean expression for this gate is Y=A*B, Where Y is the output and A,B are the inputs.

OR gate: OR gate is sometimes called the "any" or "all" gate. So the OR gate outputs "HIGH" At the output when either or both of the inputs is "HIGH". The truth table is shown in the diagram. The Boolean expression for this gate is Y=A+B, Where Y is the output and A,B are the inputs.

Ex-OR gate: The Ex-OR gate is sometimes called the "any but not all" gate so the Ex-Or gate Output HIGH at the output only if either but not both the inputs is HIGH. The Truth table is shown in the diagrams the Boolean expression for this gate is Y=AOB, Where Y is the output and A,B are the inputs.
