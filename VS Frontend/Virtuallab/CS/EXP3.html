DELAY TIME:

Delay time is the time required for the response to reach from 0% to 50% of the steady state value.

RISE TIME: :

Rise time is the time required for the response to reach from 10% to 90% of the steady state value for under-damped systems.

PEAK TIME:

Peak time is the time required to reach the response first peak of the overshoot.

PEAK OVERSHOOT:

Maximum overshoot is the maximum value of the response measured from unity.

SETTLING TIME:

Settling time is for the response to settle around the steady state value with the variation not exceeding a permissible tolerance level.
